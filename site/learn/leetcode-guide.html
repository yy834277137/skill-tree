#21 Merge Two Sorted Lists · 合并两个有序链表

链表 题目链接

将两个升序链表合并为一个新的 升序链表并返回。新链表由前面两个链表的所有结点拼接而成。

约束：两个链表的结点数目范围均为 [0, 50]；-100 <= Node.val <= 100；均按 非递减顺序排列。

示例 1

输入：list1 = [1,2,4], list2 = [1,3,4]

示例 2

输入：list1 = [], list2 = []
输出：[]

示例 3

输入：list1 = [], list2 = [0]
输出：[0]

#86 Partition List · 分隔链表

链表 题目链接

给定头结点 head 与一个值 x，将链表分隔为两部分：所有 小于 x 的结点出现在大于或等于 x 的结点之前，且两部分内结点相对顺序不变。

约束：链表中结点数目在 [0, 200]；-100 <= Node.val <= 100；-200 <= x <= 200。

示例 1

示例 2

输入：head = [2,1], x = 2
输出：[1,2]

#141 Linked List Cycle · 环形链表

链表双指针 题目链接

判断链表中是否存在环。若某结点在链表重新遍历过程中可被再次到达，则存在环。

约束：结点数 [0, 10^4]；-10^5 <= Node.val <= 10^5；pos 为 -1 或链表中的有效下标（评测用，不在函数参数里）。

示例 1

输入：head = [3,2,0,-4], pos = 1
输出：true
解释：尾结点连到下标 1（值为 2 的结点），成环。

示例 2

输入：head = [1,2], pos = 0
输出：true

示例 3

输入：head = [1], pos = -1
输出：false

#160 Intersection of Two Linked Lists · 相交链表

#378 Kth Smallest Element in a Sorted Matrix · 有序矩阵中第 K 小的元素

堆二分 题目链接

给定 n x n 矩阵，每行每列均按升序排好，返回矩阵中第 k 小的元素（非第 k 个不同值，计重）。

约束：n == matrix.length；1 <= n <= 300；-10^9 <= matrix[i][j] <= 10^9；每行每列升序；1 <= k <= n^2。

示例 1

输入：matrix = [[1,5,9],[10,11,13],[12,13,15]], k = 8
输出：13

示例 2

输入：matrix = [[-5]], k = 1
输出：-5

#373 Find K Pairs with Smallest Sums · 查找和最小的 K 对数字

堆 题目链接

给定两个以升序排列的整数数组 nums1、nums2 和整数 k，需找到和最小的 k 对 (nums1[i], nums2[j])。

约束：1 <= nums1.length, nums2.length <= 10^5；-10^9 <= nums1[i], nums2[j] <= 10^9；nums1、nums2 均为升序；1 <= k <= 10^4；k <= nums1.length * nums2.length。

示例 1

输入：nums1 = [1,7,11], nums2 = [2,4,6], k = 3
输出：[[1,2],[1,4],[1,6]]

示例 2

输入：nums1 = [1,1,2], nums2 = [1,2,3], k = 2
输出：[[1,1],[1,1]]

#2 Add Two Numbers · 两数相加

链表模拟 题目链接

两个非空链表表示两个非负整数，数字逆序存储，每位一结点。返回表示和的链表（逆序）。假定无前导零（除数字 0 本身）。

约束：结点数 [1, 100]；0 <= Node.val <= 9；除 0 外无前导零。

示例 1

输入：l1 = [2,4,3], l2 = [5,6,4]
输出：[7,0,8]
解释：342 + 465 = 807

示例 2

输入：l1 = [0], l2 = [0]
输出：[0]

示例 3

输入：l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9]
输出：[8,9,9,9,0,0,0,1]
